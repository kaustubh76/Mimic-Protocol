  WHAT IS MIRROR PROTOCOL?
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Mirror Protocol transforms on-chain trading behavior into executable,
delegatable NFTs. Successful trading patterns are detected by Envio
HyperSync (47ms), minted as ERC-721 tokens, and can be delegated to
via MetaMask Smart Accounts for automated pattern replication.

  HOW IT WORKS (4-Step Flow)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: Pattern Detection
┌─────────────────────────────────────────────────────────────────────┐
│ Trader executes successful trades                                   │
│           ↓                                                          │
│ Envio indexes events in real-time (<50ms)                           │
│           ↓                                                          │
│ PatternDetector analyzes behavioral sequence                        │
│           ↓                                                          │
│ Pattern validated & minted as NFT                                   │
└─────────────────────────────────────────────────────────────────────┘

STEP 2: Pattern Browsing
┌─────────────────────────────────────────────────────────────────────┐
│ Users browse patterns with Envio-powered metrics                    │
│ • Win rate: 65%, ROI: +23%, Volume: $1.2M                          │
│ • Real-time performance updates                                     │
│ • Creator attribution & history                                     │
└─────────────────────────────────────────────────────────────────────┘

STEP 3: Delegation Creation
┌─────────────────────────────────────────────────────────────────────┐
│ User selects pattern & creates delegation                           │
│                                                                      │
│ SIMPLE MODE:                    ADVANCED MODE:                      │
│ • Pattern NFT                   • Pattern NFT                       │
│ • Percentage (50%)              • Percentage (50%)                  │
│                                 • Max spend per tx ($1000)          │
│                                 • Daily limit ($5000)               │
│                                 • Token whitelist [USDC, WETH]      │
│                                 • Conditions: winRate > 60%         │
│                                 • Expiration: 30 days               │
└─────────────────────────────────────────────────────────────────────┘

STEP 4: Automated Execution
┌─────────────────────────────────────────────────────────────────────┐
│ Pattern creator executes trade                                      │
│           ↓                                                          │
│ Execution engine queries delegations (via Envio <50ms)              │
│           ↓                                                          │
│ Validates permissions & conditions                                  │
│           ↓                                                          │
│ Executes proportional trades for all delegators                     │
│           ↓                                                          │
│ Updates performance metrics                                         │
└─────────────────────────────────────────────────────────────────────┘

   spending limits aren't exceeded.
